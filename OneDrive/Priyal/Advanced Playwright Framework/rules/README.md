# Project Rules

Folder for enforced rules / conventions. Each `.md` file = one rule.

| Rule | File | Trigger |
|------|------|---------|
| Test Quality Checks | [test-quality-checks.md](./test-quality-checks.md) | Adding/modifying any test under `src/tests/**` |

## How rules apply

- Claude / contributors MUST read this folder before making changes.
- CI references these rules for gate checks.
- New rule → add `.md` file here + update this table.
